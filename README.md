# ph-postal-code

[![npm version](https://img.shields.io/npm/v/ph-postal-code.svg)](https://www.npmjs.com/package/ph-postal-code)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

Philippine postal/ZIP code validation and lookup

## Installation

```bash
npm install ph-postal-code
```

## Functions

- `validatePostalCode` — Validate Philippine postal code (4 digits)
- `formatPostalCode` — Format postal code with zero-padding
- `getRegion` — Get general region from postal code prefix
